Emotions Word Game

The theme of this game is "emotions" and uses words describing various emotions from The Feeling Wheel developed by Dr. Gloria Willcox. This game helps to improve emotional granularity by learning more specific words to describe emotions.

Instructions:
Your task is to find words that match the given category. Match as many as you can within 60 seconds. You have 3 tries for wrong matches.
